# Simple Coffee Listing

Aplicação de listagem de cafés com filtro por disponibilidade, desenvolvida como projeto de prática com React, TypeScript e Tailwind CSS.

## Deploy

[Acesse aqui](https://alanHsiqueira.github.io/simple-coffee-listing/)

## Funcionalidades

- Listagem de cafés com imagem, preço e avaliação
- Filtro entre todos os produtos e disponíveis
- Estado de carregamento e tratamento de erro
- Validacao de contrato da API com TypeScript type guards.

##  Tecnologias

- React
- TypeScript
- Vite
- Tailwind CSS

## Como rodar localmente
```bash
npm install
npm run dev
```
